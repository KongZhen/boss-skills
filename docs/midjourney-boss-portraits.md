## 使用说明

1. 每个prompt直接粘贴到Midjourney即可
2. 建议先生成一张测试统一风格，确认 `--s 250 --niji 6` 的效果
3. 如果需要更统一的像素颗粒感，可以在后缀加 `--tile` 或调整 `--s` 值
4. 生成后建议统一裁剪为 **512×512** 或 **256×256**，在网页上以 `image-rendering: pixelated` 渲染
5. 如果 niji 模式对8-bit像素效果不好，可替换为 `--v 6.1` 并加 `pixel art, 8-bit, NES game sprite` 强调

## 备用风格变体

如果想要更卡通/Q版的风格：
```
把 "NES retro game style" 替换为 "chibi pixel art, cute 16-bit RPG character portrait"
```

如果想要更暗黑/严肃的风格：
```
把后缀替换为 "dark pixel art portrait, SNES RPG villain select screen, dramatic lighting, CRT scanline effect"
```
